# Projeto TI - Arquitetura Multi-Container

## 🏗️ Arquitetura do Projeto

Este projeto implementa uma arquitetura completa de microserviços com:

- **Frontend**: Site institucional responsivo (HTML/CSS/JS)
- **API**: Node.js + Express + MongoDB para gerenciamento de tarefas
- **Database**: MongoDB para persistência de dados
- **Reverse Proxy**: Nginx para servir o frontend

## 📋 Pré-requisitos

- Docker e Docker Compose
- Git
- (Opcional) WSL2 para Windows
